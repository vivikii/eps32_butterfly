使用说明（必读）

1. 请用 Android 手机 Chrome 打开本页（iPhone Safari 不支持 Web Bluetooth）。
2. 须通过 HTTPS 访问（不能双击本地 html 文件）。可用 GitHub Pages 部署，或电脑运行：
npx serve web -p 3000 再用 ngrok 提供 https 链接。
3. 连接后拖滑条会实时发送参数；扑翼中也可调参。
4. 设备名：Butterfly_C3
